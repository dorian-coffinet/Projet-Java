Projet de Jeu en JAVA

Le but est de faire déplacé notre personnage avec le pave numérique à travers un labyrinthe. (Représenté par la grille de points)

Les monstres doivent bouger aléatoirement.

Le personnage et les monstres peuvent traverser le labyrinthe de haut en bas et de bas en haut.

Objectif  bonus non implémenté : se faire combattre le personne et les monstres (gain d’xp, objets etc etc)
