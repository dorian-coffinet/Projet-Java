Le projet Compilation

En se basant sur un langage inventé dénommé  LESS, ayant sa propre syntaxe et grammaire. Son format est : exemple.less
Le projet à l’heure actuelle fait :

	* L’analyse lexicale
	* L’analyse syntaxique
	* Les vérifications sémantiques.

Le projet a pour but d’aller jusqu’à la génération de code.
